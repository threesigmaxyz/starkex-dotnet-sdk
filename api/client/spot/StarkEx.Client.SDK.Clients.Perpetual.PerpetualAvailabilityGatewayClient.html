GetBatchDataAsync(Int32, CancellationToken)

Get the data availability information for a specific batch. The availability information includes the ID of the previous batch in chronological order and the list of Merkle leaves (positions and orders) that were modified in the requested batch. prev_batch_id of -1 indicates that there is no previous state. Note that prev_batch_id might differ from batch_id - 1. This happens when a previously submitted batch is rejected on-chain.
