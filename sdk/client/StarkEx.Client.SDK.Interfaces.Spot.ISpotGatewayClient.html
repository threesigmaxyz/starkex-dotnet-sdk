GetFirstUnusedTxAsync(CancellationToken)

Gets the next transaction id that all of its predecessors exist in the system. If no ids exist in the system - then 0 will be returned. While most of the time this implies the next tx_id used should be the returned value, this is not guaranteed, since while this value is returned, some new transaction may be written, rendering the returned response irrelevant.
