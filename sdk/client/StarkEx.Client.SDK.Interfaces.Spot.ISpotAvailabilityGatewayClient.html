GetBatchDataAsync(Int32, Boolean, CancellationToken)

Get the data availability information for a specific batch. The availability information includes the ID of the previous batch in chronological order and the list of tree leaves (Validium vaults tree and orders tree) that were modified in the requested batch. If validate_rollup is set to True, the list of rollup vault tree leaves that were modified is also returned. A prev_batch_id of -1 indicates that there is no previous state. Note that prev_batch_id might differ from batch_id -1. This happens when a previously submitted batch is rejected on-chain.
